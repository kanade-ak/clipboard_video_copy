# 追加メニュー
以下2つをプレビュー画面右クリック→`クリップボードにイメージをコピー`に追加します。
- `クリップボードに動画をコピー`
- `クリップボードに動画をコピー(背景透過)`

# 使い方
1:編集したり動画をタイムラインに入れる  
2:範囲選択でコピーする箇所を決める  
3:`クリップボードに動画をコピー`又は`クリップボードに動画をコピー(背景透過)`を押す  
4:貼り付ける  

# 色々なこと
- ffmpeg.exeをPATHで通すか`C:\\ffmpeg\\bin`に配置しないとエンコードできません
- `Plugin\ClipboardVideoCopy\temp`に動画が生成されます
- バージョンアップで動かなくなる可能性があります
- 使用は自己責任でお願いします
